새 버전 APK를 올렸을 때 링크 업데이트 방법

새 Release를 만든 후 이 페이지의 다운로드 링크를 수정하세요

새 버전(예: v1.1.0)을 GitHub Release에 올렸다면, 아래 두 줄만 수정하면 됩니다.

Genspark 에디터에서 deploy-guide.html 열고 Ctrl+F로 app-debug.apk 검색 → 아래 두 줄 교체:

href="...releases/download/v1.0.0/app-debug.apk"
href="...releases/download/v1.1.0/app-debug.apk"
⬆ 버전 태그 부분(v1.0.0)만 새 태그로 바꾸면 끝!

자동 빌드 → 사이트 다운로드 전체 흐름

1. GpsService.java 또는 기타 파일 수정GitHub 웹에서 직접 수정하거나 git push
2. GitHub Actions 자동 빌드 (5~7분)Actions 탭에서 🟡 → ✅ 완료 확인
3. GitHub Release에 APK 자동 게시영구 다운로드 URL 생성 (만료 없음)
4. 이 페이지에서 바로 다운로드 → 기사님 전달위 "최신 APK 다운로드" 버튼 클릭

기사님 설치 방법 요약

  • 1
    APK 다운로드 후 카카오톡으로 기사님께 전송
    카카오톡 채팅 → 파일 첨부 → APK 파일 선택
  • 2
    기사님 폰: 카카오톡 파일 클릭 → "알 수 없는 출처 허용" → 설치
    경고창 → 설정 → 이 출처 허용 ON → 뒤로가기 → 설치
  • 3
    ⚠️ 위치 권한 "항상 허용" 필수
    설정 → 앱 → 화물운송 기사 → 권한 → 위치 → "항상 허용"
    배터리 → 앱별 배터리 관리 → 화물운송 기사 → "제한 없음"
