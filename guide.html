1단계 · 룸 접속

관리자에게 받은 URL과 비밀번호로 입장

  • 1
    URL로 접속
    관리자에게 받은 링크를 클릭하거나, 메인 화면 고객사 카드를 선택 후 회사명을 찾아 클릭합니다.
  • 2
    룸 비밀번호 입력
    관리자에게 받은 룸 비밀번호를 입력하고 [입장하기]를 클릭합니다.
    한 번 입장하면 8시간 동안 자동 유지됩니다.
자주 사용한다면 브라우저에 즐겨찾기로 저장해 두세요. URL에 룸 ID가 포함되어 있어 바로 접속됩니다.

2단계 · 배송 등록

출발 전 기사 정보와 PIN을 함께 등록

  • 1
    [+ 배송 등록] 버튼 클릭
    배송 목록 우상단의 파란 버튼을 클릭합니다.
  • 2
    기본 정보 입력
    기사명 · 차량번호 · 화물 품목 · 출발지 · 목적지를 입력합니다.
    ❄️ 화물 타입을 반드시 선택하세요 — 냉동/냉장·냉동·냉장·상온
    냉동·냉장류 선택 시 온도기록지 업로드 후 AB온도 확인 안내가 기사 앱에 표시됩니다.
  • 3
    기사 PIN 설정 필수
    숫자 4~8자리로 PIN을 정합니다. 이 PIN은 기사가 로그인할 때 사용하므로 기사에게 문자나 카톡으로 전달해 주세요.
  • 4
    [등록] 클릭
    저장 즉시 배송 목록에 나타납니다. 기사에게 이름 + PIN을 알려주세요.
PIN은 저장 후 확인이 불가능합니다. 기사에게 전달 전 메모해 두거나 카톡으로 먼저 보내 주세요.

🔔 실시간 알림

기사 접속·상차완료·하차완료 시 자동 알림

  • 🚛
    기사 앱 접속
    기사가 앱에 로그인해 업무를 시작하면 화주 룸 화면 오른쪽 상단 🔔 벨 아이콘에 알림이 표시됩니다.
  • 📦
    상차완료
    기사가 [상차 완료 처리] 버튼을 누르면 알림이 발송됩니다. 배송 목록도 자동으로 갱신됩니다.
  • ✅
    하차완료
    모든 하차 완료 처리 시 알림이 발송됩니다. 벨 아이콘 클릭 시 알림 목록을 확인할 수 있습니다.
알림은 10초 간격으로 자동 갱신됩니다. 페이지를 열어두면 자동으로 최신 알림을 받습니다.

📸 기사에게 사진 촬영 요청

서류 미업로드 시 기사 앱으로 직접 요청

  • 1
    배송 상세 열기
    배송 목록에서 해당 배송 카드를 클릭하면 상세 정보가 열립니다.
  • 2
    📸 촬영 요청 버튼 클릭
    사진이 업로드되지 않은 항목(거래명세표/온도기록지)에 📸 촬영 요청 버튼이 표시됩니다. 클릭하면 기사 앱에 알림 배너가 뜹니다.
  • 3
    기사가 촬영 후 자동 업로드
    기사가 촬영하면 즉시 업로드되어 배송 상세에서 바로 확인할 수 있습니다.
촬영 요청은 배송이 운송 중인 상태에서만 가능합니다. 하차완료 배송은 요청할 수 없습니다.

배송 탭

  • ①
    배송 카드를 클릭하면 상세 정보(기사·차량·화물·상태·GPS·서류)를 확인할 수 있습니다.
  • ②
    카드 하단 아이콘으로 서류 업로드 여부를 한눈에 확인: 파란색=업로드됨 회색=미업로드
  • ③
    경유 배송은 경유N 배지로 몇 곳을 경유하는지 표시됩니다.

지도 탭

  • ①
    기사가 GPS를 켜면 실시간 위치가 지도에 트럭 아이콘으로 표시됩니다.
  • ②
    초록 아이콘 = 5분 이내 최신 위치 / 회색 아이콘 = 위치 오래됨. 30초마다 자동 갱신됩니다.

서류 탭

  • ①
    상단 드롭다운에서 배송건을 선택하면 해당 건의 모든 서류 사진이 표시됩니다.
  • ②
    경유 배송은 하차지별로 구분되어 표시됩니다. 사진 클릭 시 확대 가능합니다.

배송 삭제

잘못 등록한 배송 건 삭제 방법

  • 1
    배송 카드 클릭 → 상세 모달 하단 삭제 버튼 클릭
  • 2
    확인 팝업에서 [확인]을 누르면 즉시 삭제됩니다.
삭제된 배송은 복구되지 않습니다. 업로드된 사진도 함께 삭제됩니다.

1단계 · 로그인

고객사에서 받은 이름과 PIN으로 접속

  • 1
    기사 업무 화면 접속
    메인 화면에서 기사 카드를 클릭하거나, 관리자·화주에게 받은 QR 코드를 스캔합니다.
  • 2
    이름 + PIN 입력
    화주가 배송 등록 시 설정한 기사 이름과 PIN 코드를 입력합니다. 이름은 공백 없이 정확히 일치해야 합니다.
  • 3
    배송건 선택 → [업무 시작]
    내 이름에 등록된 배송 목록이 표시됩니다. 해당 배송건을 선택하고 [업무 시작]을 누르면 GPS가 자동으로 켜집니다.
로그인 상태는 12시간 유지됩니다. 앱을 닫았다 열어도 자동으로 이어집니다.

2단계 · 상차 완료 처리

서류 촬영 → 자동 업로드 → 상차 완료

  • 1
    거래명세표 촬영
    촬영/선택 버튼을 눌러 카메라로 촬영하거나 갤러리에서 선택합니다. 선택 즉시 자동으로 업로드됩니다.
  • 2
    온도기록지 촬영
    같은 방법으로 온도기록지도 촬영합니다. 업로드 완료 후 수정 버튼이 나타납니다.
  • 3
    [상차 완료 처리] 클릭
    노란색 버튼을 눌러 상차 완료를 처리합니다. 이때부터 배송 상태가 운송중으로 변경됩니다.
잘못 촬영했으면 [수정] 버튼을 눌러 다시 찍으면 됩니다. 기존 사진을 덮어씁니다.

3단계 · 하차 완료 처리

목적지 도착 후 서류 촬영 및 완료 처리

  • 1
    하차 1 섹션 확인
    상차 섹션 아래에 하차 1 섹션이 자동으로 생성되어 있습니다.
  • 2
    거래명세표 · 온도기록지 촬영
    상차와 동일하게 촬영 즉시 자동 업로드됩니다.
  • 3
    [하차 1 하차 완료 처리] 클릭
    초록색 버튼을 클릭합니다. 모든 하차가 완료되면 배송 상태가 하차완료로 변경되고 GPS가 자동으로 종료됩니다.

경유 하차 처리 (여러 목적지)

1건의 배송에서 여러 곳에 하차할 때

  • 1
    [+ 경유 하차지 추가] 클릭
    하차 섹션 아래 파란색 점선 버튼을 클릭하면 하차 2, 하차 3... 세트가 추가됩니다.
  • 2
    하차지 이름 변경 (선택)
    입력란에 "1차 납품처", "ABC마트" 등 구체적인 이름으로 변경할 수 있습니다.
  • 3
    각 하차지별 서류 촬영 → 하차 완료
    각 하차지에 도착할 때마다 사진 촬영 후 [하차 완료 처리]를 클릭합니다. 모든 하차지 완료 시 GPS 자동 종료.
잘못 추가한 경유지는 우측 상단 [X 삭제] 버튼으로 제거할 수 있습니다.

GPS 위치 공유

화주가 실시간으로 위치를 확인합니다

  • ✓
    업무 시작 시 자동 켜짐
    [업무 시작] 클릭 시 GPS가 자동으로 활성화됩니다. 브라우저 위치 권한을 허용해 주세요.
  • ✓
    하차 완료 전 GPS 유지
    모든 하차지 완료 전까지는 GPS를 끌 수 없습니다. 신호가 끊기면 5초 후 자동 재연결됩니다.
  • ✓
    하차 완료 시 자동 종료
    모든 하차지 완료 처리 후 GPS가 자동으로 꺼집니다.
네비게이션 사용 시: 상차완료 버튼을 누르면 앱이 자동으로 백그라운드로 전환됩니다. 네비 앱을 켜고 출발하면 됩니다.
Android APK 설치 시: 백그라운드 GPS가 완전히 유지됩니다. 위치 권한을 "항상 허용"으로 설정하세요. 배터리 최적화도 "제한 없음"으로 설정하면 더 안정적입니다.
재접속 자동 복원: 앱이 꺼졌어도 12시간 이내 재접속하면 이전 업무와 GPS가 자동으로 복원됩니다.

  • 2
    화주에게 전달
    생성된 룸의 URL과 비밀번호를 화주 담당자에게 전달합니다. QR 페이지에서 QR 코드로 공유할 수도 있습니다.
  • 3
    수정 · 삭제
    룸 목록에서 수정 또는 삭제 버튼으로 관리합니다.

  • ②
    배송 상태를 직접 변경하거나 필요시 삭제할 수 있습니다.

사진 보관함

날짜·거래처별 서류 사진 통합 조회

  • 1
    접속 방법
    관리자 헤더 사진 보관함 클릭 또는 메인 화면 하단 카드 클릭
  • 2
    필터 활용
    거래처·날짜 범위·기사명·서류 유형으로 검색 가능합니다.
  • 3
    보기 방식 선택
    날짜별 · 거래처별 · 전체 그리드 세 가지 뷰를 제공합니다.

Android APK 빌드 · GPS 주기 변경 · 재배포 절차

  • 1
    웹(HTML) 변경 → Genspark 재배포만
    driver.html·CSS 등 웹 파일 변경 시 Genspark Publish 탭에서 재배포하면 됩니다. 기사님 APK 재설치 불필요.
  • 2
    Java 코드 변경 → GitHub push → APK 재빌드
    GpsService.java(GPS 주기·알림 문구)·AndroidManifest.xml 변경 시 GitHub에 push하면 Actions가 자동 빌드합니다. APK 다운로드 후 기사님께 전송.
  • 3
    자세한 단계별 가이드
    배포 가이드 열기

📷 이미지 서버 설정 (중요)

500 에러 방지 — UVIS 서버 계정 등록 필수

기사님 사진이 DB에 누적되면 500 Internal Server Error가 발생합니다. 반드시 아래 설정을 완료하세요.
  • 1
    설정 탭 이동
    admin.html 로그인 후 설정 탭 클릭
  • 2
    UVIS 계정 입력 및 저장
    UVIS 아이디와 비밀번호를 입력하고 계정 저장 클릭

자주 묻는 질문

사용 중 불편하신 점을 먼저 확인해 보세요

🔑 PIN을 잊어버렸어요 / PIN이 틀렸다고 나와요
PIN은 한 번 저장되면 관리자도 확인할 수 없습니다. 화주(고객사) 담당자에게 연락해서 배송을 삭제하고 새로 등록(새 PIN 설정)해 달라고 요청하세요.
🚛 배송건이 목록에 안 보여요
① 이름 확인: 화주가 등록한 기사 이름과 정확히 일치해야 합니다. 공백, 오탈자를 확인하세요.
② 이미 완료됨: 하차완료 처리된 배송은 기본적으로 목록 맨 아래에 ✅ 표시와 함께 보입니다.
③ PIN 불일치: PIN이 맞지 않으면 해당 배송이 조회되지 않습니다.
📸 사진이 업로드가 안 돼요
① 네트워크 확인: 인터넷이 연결되어 있는지 확인하세요.
② 카메라 권한: 브라우저에서 카메라 접근이 허용되어 있는지 확인하세요.
③ 재시도: 업로드 실패 시 토스트 메시지가 나타납니다. [수정] 버튼을 눌러 다시 촬영하면 됩니다.
④ 파일 크기: 이미지는 자동으로 1024px 이하로 압축됩니다.
📍 GPS가 작동하지 않아요
① 위치 권한: 브라우저에서 위치 접근 권한을 허용으로 설정해야 합니다. (iOS: 설정 → Safari → 위치, Android: 크롬 설정 → 사이트 설정 → 위치)
② 실내 환경: 실내에서는 GPS 신호가 약합니다. 신호 약함 시 5초 후 자동 재시도합니다.
③ 헤더 표시: 화면 상단에 GPS 켜짐 표시가 있어야 정상입니다.
🗺️ 지도에 기사 위치가 안 보여요
① GPS 미활성: 기사가 GPS를 켜지 않았거나 권한이 거부된 경우입니다.
② 배송 상태: 운송중(transit) 또는 상차완료(loading) 상태일 때만 지도에 표시됩니다.
③ 갱신 주기: 30초마다 위치가 서버에 전송됩니다. 자동 갱신 토글을 켜두면 30초마다 지도가 새로고침됩니다.
🏢 고객사 룸 비밀번호를 바꾸고 싶어요
관리자 페이지(admin.html) → 고객사 관리 탭 → 해당 룸의 [수정] 버튼 클릭 → 새 비밀번호 입력 후 저장하면 됩니다. 변경 후 화주에게 새 비밀번호를 전달하세요.
🔄 화면이 바뀌지 않거나 오래된 화면이 보여요
브라우저 캐시 문제입니다. 강제 새로고침을 해주세요.
• PC: Ctrl + Shift + R (Windows) / Cmd + Shift + R (Mac)
• 모바일: 브라우저 설정 → 캐시 삭제 후 재접속
📂 서류 사진을 다운로드하고 싶어요
사진 보관함(gallery.html) 또는 서류 탭에서 사진을 꾹 눌러서(모바일) 또는 우클릭(PC) → 이미지 저장으로 개별 다운로드할 수 있습니다.
🔴 500 에러가 발생해요 (데이터를 불러오지 못함)
기사님 사진이 DB에 직접 저장되어 서버 메모리를 초과한 상태입니다.

해결 방법:
① 관리자(admin.html) → 설정 탭 이동
② UVIS 아이디·비밀번호 입력 → 계정 저장
③ 연결 테스트 클릭 → ✅ 성공 확인
④ 기사용 링크 생성 → 카카오톡으로 기사님께 전송
⑤ 기사님이 해당 링크로 접속하면 설정 완료

이후 신규 사진은 rhkdtls.cloud MinIO에 저장되어 500 에러가 사라집니다.
📱 기사님이 네비게이션 쓸 때 GPS가 끊겨요
상차완료 버튼을 누르면 앱이 자동으로 백그라운드로 전환됩니다.

추가 설정:
① 위치 권한: "항상 허용"으로 설정 (한 번 허용 X)
② 배터리 최적화: 설정 → 앱 → 화물운송기사 → "제한 없음"
③ Android APK를 설치하면 백그라운드 GPS가 완전히 유지됩니다.
④ 앱이 꺼졌어도 12시간 이내 재접속하면 자동 복원됩니다.
