배송 상태: 대기중 → 상차완료 → 운송중 → 하차완료
1단계 · 룸 접속
관리자에게 받은 URL과 비밀번호로 입장
1
URL로 접속
관리자에게 받은 링크 클릭 또는, 메인 화면 고객사 카드 → 회사명 선택.
2
룸 비밀번호 입력
관리자에게 받은 비밀번호 입력 → [입장하기]. 한 번 입장하면 8시간 자동 유지됩니다.
자주 사용한다면 브라우저에 즐겨찾기로 저장해 두세요.
2단계 · 배송 등록
출발 전 기사 정보와 PIN을 함께 등록
1
[+ 배송 등록] 버튼 클릭
배송 목록 우상단의 파란 버튼을 클릭합니다.
2
기본 정보 입력
기사명 · 차량번호 · 화물 품목 · 출발지 · 목적지를 입력합니다.
3
기사 PIN 설정 필수
숫자 4~8자리 PIN. 기사에게 문자·카톡으로 전달하세요.
4
[등록] 클릭
저장 즉시 배송 목록에 나타납니다.
PIN은 저장 후 확인 불가능합니다. 기사에게 전달 전 메모해 두거나 카톡으로 먼저 보내주세요.
3단계 · 배송 현황 확인
배송 목록 · GPS 지도 · 서류 사진 탭 활용
배송 탭
• 배송 카드 클릭 → 상세 정보(기사·차량·화물·상태·GPS·서류) 확인
• 카드 하단 아이콘: 파란색=서류있음 회색=없음
• 카드의 📍위치추적 버튼 클릭 시 지도 탭으로 즉시 이동!
지도 탭
• 기사 GPS 실시간 확인. 초록=5분 이내, 회색=오래됨. 30초 자동 갱신.
서류 탭
• 드롭다운에서 배송건 선택 → 상·하차 사진 확인. 사진 클릭 시 확대 가능.
1단계 · 로그인
고객사에서 받은 이름과 PIN으로 접속
1
기사 업무 화면 접속
메인 화면 기사 카드 클릭 또는 QR 코드 스캔.
2
이름 + PIN 입력
화주가 등록한 기사 이름과 PIN을 입력합니다. 이름은 공백 없이 정확히 일치해야 합니다.
3
배송건 선택 → [업무 시작]
배송건 선택 후 [업무 시작] → GPS가 자동으로 켜집니다.
로그인 상태는 12시간 유지됩니다. 앱을 닫았다 열어도 자동으로 이어집니다.
2단계 · 상차 완료 처리
서류 촬영 → 자동 업로드 → 상차 완료
1
거래명세표 촬영
[촬영/선택] 버튼 → 카메라 촬영 또는 갤러리 선택. 선택 즉시 자동 업로드됩니다.
2
온도기록지 촬영
동일하게 촬영. 업로드 완료 후 주황색 [수정] 버튼 표시.
3
[상차 완료 처리] 클릭
노란색 버튼 클릭 → 배송 상태가 운송중으로 변경됩니다.
잘못 촬영했으면 [수정] 버튼을 눌러 다시 찍으면 됩니다.
3단계 · 하차 완료 처리
목적지 도착 후 서류 촬영 및 완료 처리
1
하차 1 섹션 확인
상차 섹션 아래에 하차 1 섹션이 자동 생성되어 있습니다.
2
거래명세표·온도기록지 촬영
상차와 동일하게 촬영 즉시 자동 업로드됩니다.
3
[하차 완료 처리] 클릭
초록 버튼 클릭. 모든 하차 완료 시 배송완료로 변경되고 GPS가 자동 종료됩니다.
경유 하차 처리 (여러 목적지)
1건 배송에서 여러 곳에 하차할 때
1
[+ 경유 하차지 추가] 클릭
하차 섹션 아래 파란 점선 버튼 → 하차 2, 3... 추가됩니다.
2
하차지 이름 변경 (선택)
"1차 납품처", "ABC마트" 등 구체적인 이름으로 변경 가능합니다.
3
각 하차지별 처리
각 하차지 도착 시 사진 촬영 후 [하차 완료]. 모두 완료 시 GPS 자동 종료.
잘못 추가한 경유지는 우측 상단 [X 삭제] 버튼으로 제거할 수 있습니다.
GPS 위치 공유
화주가 실시간으로 위치를 확인합니다
✅ 업무 시작 시 GPS 자동 켜짐 — 브라우저 위치 권한을 허용해 주세요.
✅ 모든 하차지 완료 전까지 GPS 유지 — 신호 끊기면 5초 후 자동 재연결.
✅ 하차 완료 시 GPS 자동 종료.
화면을 꺼도 GPS는 계속 작동합니다. 브라우저를 백그라운드로 유지해 주세요.
관리자 로그인
admin.html — 비밀번호로 접속
메인 화면 관리자 카드 클릭 → 비밀번호 입력. 로그인 후 4시간 세션 유지됩니다.
고객사 룸 관리
화주별 룸 생성·수정·삭제
1
룸 생성
[+ 룸 추가] → 룸 이름(회사명)·비밀번호·설명·연락처 입력 → [저장]
2
화주에게 전달
생성된 룸의 URL과 비밀번호를 화주 담당자에게 전달. QR 페이지에서 QR 공유 가능.
3
수정 · 삭제
룸 목록에서 수정/삭제 버튼으로 관리합니다.
사진 보관함
날짜·거래처별 서류 사진 통합 조회
관리자 헤더 [사진 보관함] 클릭 또는 메인 화면 사진 보관함 카드 클릭.
거래처·날짜 범위·기사명·서류 유형으로 검색. 날짜별·거래처별·전체 그리드 3가지 뷰.

자주 묻는 질문

🔑 PIN을 잊어버렸어요 / PIN이 틀렸다고 나와요
PIN은 저장 후 확인 불가능합니다. 화주(고객사) 담당자에게 연락해서 배송을 삭제하고 새로 등록(새 PIN 설정)해 달라고 요청하세요.
🚛 배송건이 목록에 안 보여요
① 이름 확인: 화주가 등록한 이름과 정확히 일치해야 합니다.
② 이미 완료됨: 하차완료 배송은 목록 맨 아래에 ✅ 표시로 보입니다.
③ PIN 불일치: PIN이 맞지 않으면 조회되지 않습니다.
📸 사진이 업로드가 안 돼요
① 인터넷 연결 확인
② 브라우저 카메라 권한 허용 확인
③ 실패 시 토스트 메시지 확인 후 [수정] 버튼으로 재촬영
📍 GPS가 작동하지 않아요
① 위치 권한 허용 확인 (iOS: 설정→Safari→위치, Android: 크롬설정→사이트설정→위치)
② 실내 환경: 신호 약함 → 5초 후 자동 재시도
③ 헤더에 GPS 켜짐 표시 확인
🗺️ 지도에 기사 위치가 안 보여요
① 기사가 GPS를 켜지 않았거나 권한 거부된 경우
② 운송중 또는 상차완료 상태일 때만 지도에 표시됩니다.
③ 30초마다 갱신 — 자동 갱신 토글을 켜두세요.
🏢 고객사 룸 비밀번호를 바꾸고 싶어요
관리자 페이지 → 고객사 관리 탭 → 해당 룸 [수정] 클릭 → 새 비밀번호 입력 후 저장. 변경 후 화주에게 새 비밀번호를 전달하세요.
🔄 화면이 바뀌지 않거나 오래된 화면이 보여요
브라우저 캐시 문제입니다. 강제 새로고침:
• PC: Ctrl+Shift+R (Win) / Cmd+Shift+R (Mac)
• 모바일: 브라우저 설정 → 캐시 삭제 후 재접속
📂 서류 사진을 다운로드하고 싶어요
사진 보관함 또는 서류 탭에서 사진을 꾹 눌러서(모바일) 또는 우클릭(PC) → 이미지 저장으로 다운로드할 수 있습니다.
