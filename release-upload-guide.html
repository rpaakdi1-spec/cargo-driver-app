자주 묻는 질문

APK 파일이 없는데 어떻게 만드나요?

GitHub Actions로 자동 빌드하거나, Android Studio에서 직접 빌드합니다.
Actions 자동 빌드: 코드를 push하면 약 5~7분 후 Actions 탭 → Artifacts에서 APK를 다운로드할 수 있습니다.
그 APK 파일을 받아서 여기서 Release에 첨부하면 됩니다.

배포 가이드에서 "APK를 찾을 수 없습니다"가 떠요

Release에 APK 파일이 첨부되지 않은 경우입니다.
해당 Release를 열어서 Edit release → 파일 다시 첨부 → Update release로 수정하세요.

새 버전을 올리려면 어떻게 하나요?

새 Release를 만들면 됩니다. 태그를 v1.1.0으로 올려서 새 Release를 만들고 새 APK를 첨부하세요.
배포 가이드 페이지는 항상 가장 최신 Release의 APK를 자동으로 표시합니다.

"Set as pre-release" 체크해야 하나요?

체크하지 마세요. pre-release로 표시하면 배포 가이드의 releases/latest API에서 감지되지 않습니다.
기본값(체크 안 함)으로 두고 Publish 하세요.
